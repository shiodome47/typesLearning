TypeScriptの第一歩

「箱の形」を厳密に決める：
基本の型 (Primitive Types)

変数や関数に : string のように「型のアノテーション（注釈）」をつけることで、TypeScriptは「その箱には何が入るべきか」を理解し、間違ったデータが混入するのを未然に防いでくれます。

1. 変数への代入チェック

下のボタンを押して、変数（箱）に新しい値を代入してみましょう。
指定された「型」と違うものを入れようとするとどうなるでしょうか？

: string (文字列)
let username
"Alice"
Type 'number' is not assignable to type 'string'.
: number (数値)
let age
25
Type 'string' is not assignable to type 'number'.
: boolean (真偽値)
let isLoggedIn
false
Type 'string' is not assignable to type 'boolean'.

関数を呼び出す
function greet(
  name: string ← 入口(引数)の型
): string ← 出口(戻り値)の型 {

}
