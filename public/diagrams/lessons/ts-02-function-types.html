TypeScript 関数のルール

入口と出口の厳格なチェック：
引数 と void 型

JavaScriptでは引数を渡し忘れても無視されますが、TypeScriptでは「数が合わない呼び出し」をエラーとして防ぎます。
省略しても良い引数には ? を付け、何も返さない関数には void を指定します。

2. 値を返さない void 型

void型の意味
「この関数は中で処理をして終わりで、外には何も結果を返しませんよ」という宣言です。
これにより、間違って戻り値を変数に代入しようとするミスを防げます。
function logMessage(message): void {
  console.log(message);
}
