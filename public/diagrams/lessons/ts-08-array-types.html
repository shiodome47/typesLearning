map
function getTitles(todos) {
  return todos.map(todo =>
    todo.title
  );
}

「変換」します。配列のすべての要素を1つずつ加工し、必ず同じ長さの新しい配列を作ります。今回はオブジェクトから文字列だけを抽出しています。

filter
function getActive(todos) {
  return todos.filter(todo =>
    !todo.done
  );
}

「抽出（ふるい分け）」します。条件式が true になる要素だけを残し、新しい配列を作ります。弾かれた要素は消えるため、長さは減る可能性があります。

find
function findById(todos, id) {
  return todos.find(todo =>
    todo.id === id
  );
}

「検索」します。条件に合う最初の1つを見つけた瞬間、ループを打ち切ってその要素（配列ではなく単体）を返します。見つからなければ undefined を返します。
