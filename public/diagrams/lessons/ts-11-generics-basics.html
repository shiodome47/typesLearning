TypeScript 型の柔軟性

「型の変数」で汎用的にする：
ジェネリクス (Generics)

<T> は、一時的な「型の変数（空き箱）」です。
使う時になって初めて「今回は string ね！」と型が決定し、関数やオブジェクト全体にその型が自動的に伝わっていく強力な仕組みです。

1. 型が伝播する汎用関数 (identity)

引数を渡す
function identity
function identity <T> ( value: T ): T {
return value;
}

2. 任意の型をラップする (ApiResponse)

データを渡して包む
型の設計図
type ApiResponse<T> = {
  success: boolean;
  data: T;
  error?: string;
};
function wrap
function wrap<T>(data: T): ApiResponse<T> {
  return {
    success: true,
    data // data: data の省略形
  };
}
完成したオブジェクト
型: ApiResponse<string[]>
{
  success: true,
  data: ["Alice", "Bob"]
}
