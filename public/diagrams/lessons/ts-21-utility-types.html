TypeScript 型変換の魔法

原本を加工して使い回す：
ユーティリティ型 (Utility Types)

Pick、Omit、Partial などの「便利な道具」を使うと、巨大なベース型（原本）から必要な部分だけを抽出したり、必須を任意に変えたりして、新しく専用の型を安全に作り出すことができます。

原本 (Base Type)
type User = {
id number
name string
email string
password string
createdAt Date
};

原本から作りたい型を選んでください
