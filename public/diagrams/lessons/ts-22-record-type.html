Record 型 の図解

Record<Keys, Type> は、指定したすべての キー（Keys） に対して、必ず指定した 型（Type） の値を持たせることを強制する型です。「書き忘れ」や「タイポ」を完全に防ぎます。

1. 型の強制力（なぜRecordを使うのか？）

第1引数：許されるキー (Keys)
Status
"idle" "loading" "success" "error"
Record<Status, string>
第2引数：値の型 (Type) ＝ string
const statusLabels: Record<Status, string> = {
idle: "待機中",
loading: "読込中",
success: "成功",
error: "エラー"
Property 'error' is missing in type '{ ... }' but required in type 'Record<Status, string>'.
};
