Mapped Types

[K in keyof T] という構文は、JavaScriptの for...in ループの「型バージョン」です。元の型のプロパティを1つずつ取り出し、まとめて readonly や ?（任意）を付けた「新しい型」を自動生成します。

2. 作成した型の動作確認

const ro: MyReadonly<User> = {
  id: 1,
  name: "Alice",
  email: "a@b.com"
};

ro.name = "Bob";
TypeScript Error
Cannot assign to 'name' because it is a read-only property.
すべてのプロパティに readonly が付与されたため、値の変更をコンパイル時にブロックします。
const p: MyPartial<User> = {
  id: 1
  /* name は省略 */
  /* email は省略 */
};
エラーは出ません！
省略可能 (Optional) の力
すべてのプロパティに ? が付与されたため、name や email を書かなくても、エラーにならず安全にオブジェクトを作成できます（APIの更新処理などに便利です）。
